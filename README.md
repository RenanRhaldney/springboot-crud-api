# CRUD Spring Boot

Projeto de estudo desenvolvido com:

- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- Maven
- Bean Validation
- REST API

## Funcionalidades

- Criar produto
- Listar produtos
- Buscar por ID
- Atualizar produto
- Excluir produto

## Tecnologias

- Java
- Spring Boot
- PostgreSQL
- Maven
